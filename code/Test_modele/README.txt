Petit projet prêt à lancer dans Spyder pour comprendre la classification de texte avec BERT/DistilBERT.

Commence par lire ETAPES_SPYDER.txt.

Résumé rapide :

1. Installer les dépendances :
   pip install -r requirements.txt

2. Lancer generate_data.py

3. Lancer train_model.py

4. Lancer predict.py

Le modèle entraîné sera sauvegardé dans le dossier final_model.
